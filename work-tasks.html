Copilot Tasks — For Work

This prototype shows what happens when autonomous AI (Tasks) meets enterprise context.

📋 Plan visibility — user sees every step before execution
🔒 Compliance-first — transparent data access + DLP enforcement
🔗 Cross-app orchestration — Outlook + Teams + Excel + Word + Calendar in one flow
📎 Provenance — every claim traces to a specific source
✋ Steerability — user approves before external actions
📊 Change detection — AI flags when data has shifted since last discussion
🔁 Recurring autonomy — set it once, it works every week
📝 Audit trail — every access and action logged for enterprise trust

The gap between Personal Tasks and Work Tasks is the enterprise experience layer — compliance, provenance, multi-stakeholder awareness, and audit.

Natural Language Task Creation
User describes what they want done — no forms, no templates. Same as Personal Tasks but scoped to work context.
Visible Plan with Sources
Each step shows which M365 app it accesses. The user sees the full plan before anything executes. Personal Tasks uses a browser; Work Tasks orchestrates across your org's apps.
Enterprise Compliance Surface
This doesn't exist in Personal Tasks. Work Tasks must show what data it accesses, confirm DLP compliance, and get user consent before touching org data.
Cross-App Reasoning
Copilot correlates across Outlook, Teams, and Excel — finding that a Teams thread answers an Outlook email. This cross-artifact intelligence is uniquely possible in the enterprise graph.
Provenance + Steerability
Every bullet traces to a specific file, cell, or thread. The user approves before sending to a VP. "It's not autopilot — it's a copilot" applies doubly in enterprise.
Change-Aware Meeting Prep
AI doesn't just list your meetings — it detects that data has changed since the last discussion and prepares you for the conversation. This requires persistent memory + cross-session awareness.
Audit Trail + Recurring Autonomy
Every data access is logged. The task runs autonomously every Monday. Enterprise trust requires both: the autonomy of Tasks with the accountability of an audit trail.
