Chat History Flow

A hiring plan that starts in Copilot and flows through Word, Excel, and PowerPoint

Product Decisions Demonstrated

1From chat → file: Clicking a file link in any Copilot chat opens that chat in the file's agent mode pane. No exceptions.
2File opened directly: Shows the app landing page. Hamburger shows all chats. New message = new chat.
3All chats in history: Same list as Copilot App. Not filtered by app.
4Close and reopen: Shows exactly what was showing before.
5One chat, everywhere: Stored once, visible in every surface.
Press Space, → or click to advance each step. Each step waits for you.
Prototype by Hema Sridharan · March 2026
STARTING POINTYou're in the Copilot App with sidebar. Ask about the hiring plan.
DECISION 1Copilot found a file. The file link will open Excel with this exact chat in the agent mode pane.
HANDOFF → EXCELExcel opened. Sidebar gone — now it's a document + chat pane. The chat came with it. Same conversation.
DECISION 1 — CROSS-APPFrom Excel, Copilot generated a Word doc. Clicking the link → Word with this same chat.
HANDOFF → WORDThird surface. Same chat. Copilot App → Excel → Word.
DECISION 1 — AGAINUpdated a PPT deck. Clicking → PowerPoint with same chat.
HANDOFF → POWERPOINTFourth surface, one conversation. Copilot App → Excel → Word → PPT.
DECISION 2Opened a file directly — not from a chat. Landing page: "Let's edit your spreadsheet." Fresh start.
DECISION 3Hamburger shows all your Copilot chats. Same list as the Copilot App sidebar. Not "Chats in Excel."
DECISION 5"Hiring Plan Pipeline Review" — one chat, stored once, visible everywhere. Click to continue it.
